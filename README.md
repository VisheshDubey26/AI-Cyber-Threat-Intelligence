# AI-Powered Cyber Threat Intelligence Platform

## Project Overview
This project is a web-based cybersecurity platform designed to detect malicious URLs using Artificial Intelligence and Cyber Threat Intelligence techniques.

The system analyzes URLs using multiple threat indicators and predicts whether the URL is safe or malicious.

## Key Features
- URL Threat Detection
- AI-Based Threat Prediction
- Cyber Threat Intelligence Dashboard
- Threat Database Storage
- Security Report Generation

## Technology Stack
Frontend: React.js  
Backend: Node.js with Express  
Database: MongoDB  
AI Model: Python with Scikit-learn

## Project Modules
Frontend – User interface for scanning URLs and displaying results  
Backend – API handling requests and system logic  
AI Model – Machine learning model for threat detection  
Database – Stores scanned URLs and threat results  
Documentation – Project reports and system diagrams
